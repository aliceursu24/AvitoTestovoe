## Инструкция по запуску автотестов:

Убедитесь, что у вас установлен Python 3.8+

python --version

Перейдите в директорию проекта

cd path/to/project

Установите необходимые зависимости

pip install pytest requests

Запустить тесты с помощью команды

pytest -v
